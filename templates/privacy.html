Документы сервиса

Политика конфиденциальности

Настоящий документ описывает базовые принципы обработки персональных данных и иной информации в рамках использования платформы CyberLearn Витте. Документ носит информационный характер и может обновляться по мере развития сервиса.

Общие положения

Цели обработки данных

Платформа предназначена для организации корпоративного обучения по информационной безопасности, учёта прогресса и проверки знаний. Данные пользователей обрабатываются в объёме, необходимом для предоставления этих функций: регистрация и вход, работа в личном кабинете, прохождение материалов и тестов, обратная связь с администрацией.

Состав данных

Какие сведения могут обрабатываться

  • учётные данные (например, имя пользователя и адрес электронной почты, указанные при регистрации);
  • служебная информация о работе в сервисе (прогресс по материалам, результаты тестов, время активности);
  • содержание сообщений, направленных через формы обратной связи;
  • технические данные, необходимые для стабильной работы сайта (например, сведения о сессии в рамках стандартных механизмов веб-приложений).

Меры защиты

Безопасность и ограничение доступа

Доступ к персональным и учебным данным внутри сервиса разграничивается по ролям пользователей. Администраторы и кураторы получают доступ только к тем сведениям, которые необходимы для управления обучением и поддержки пользователей. Рекомендуется использовать устойчивые пароли и не передавать учётные данные третьим лицам.

Вопросы

Связь по вопросам данных

Уточнения можно направить через раздел обратной связи или по контактам на странице «Контакты». Также доступны пользовательское соглашение и раздел «Безопасность».
